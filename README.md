**–í–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤–∏–µ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–æ–≤ (–ø–æ—Ç–æ–∫ –¥–∞–Ω–Ω—ã—Ö):** –ü–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å –æ—Ç–ø—Ä–∞–≤–ª—è–µ—Ç —Å–æ–æ–±—â–µ–Ω–∏–µ –±–æ—Ç—É -> –±–æ—Ç —á–µ—Ä–µ–∑ API –ø–µ—Ä–µ–¥–∞–µ—Ç –µ–≥–æ backend -> backend –∑–∞–ø—Ä–∞—à–∏–≤–∞–µ—Ç RAG –∏ –ø–æ–ª—É—á–∞–µ—Ç –∫–æ–Ω—Ç–µ–∫—Å—Ç -> backend –∑–∞–ø—Ä–∞—à–∏–≤–∞–µ—Ç LLM, –ø–æ–ª—É—á–∞–µ—Ç –æ—Ç–≤–µ—Ç -> –ø—Ä–∏ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ—Å—Ç–∏ backend –≤—ã–∑—ã–≤–∞–µ—Ç –≤–Ω–µ—à–Ω–µ–µ –¥–µ–π—Å—Ç–≤–∏–µ -> –æ—Ç–≤–µ—Ç –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç—Å—è –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—é —á–µ—Ä–µ–∑ –±–æ—Ç–∞. –í—Å–µ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç—ã –ª–æ–≥–∏—Ä—É—é—Ç —Å–≤–æ–∏ –¥–µ–π—Å—Ç–≤–∏—è –∏ –æ—Ç—Å—ã–ª–∞—é—Ç –º–µ—Ç—Ä–∏–∫–∏. –¢–∞–∫–∞—è —Ü–µ–ø–æ—á–∫–∞ –ø–æ–∑–≤–æ–ª—è–µ—Ç —Ä–∞–∑–¥–µ–ª–∏—Ç—å –æ—Ç–≤–µ—Ç—Å—Ç–≤–µ–Ω–Ω–æ—Å—Ç—å: RAG –æ–±–µ—Å–ø–µ—á–∏–≤–∞–µ—Ç –∑–Ω–∞–Ω–∏–µ, LLM ‚Äì —è–∑—ã–∫–æ–≤–æ–µ –æ–±—â–µ–Ω–∏–µ, –∞ backend ‚Äì –ª–æ–≥–∏–∫—É –∏ –∫–æ–Ω—Ç—Ä–æ–ª—å.

–ê—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–∞ —Å–ø—Ä–æ–µ–∫—Ç–∏—Ä–æ–≤–∞–Ω–∞ —Å —Ä–∞—Å—á–µ—Ç–æ–º –Ω–∞ **–º–∞—Å—à—Ç–∞–±–∏—Ä—É–µ–º–æ—Å—Ç—å**: –∫–∞–∂–¥—ã–π –∫–æ–º–ø–æ–Ω–µ–Ω—Ç (–±–æ—Ç, backend, RAG, LLM) –º–æ–∂–µ—Ç –±—ã—Ç—å –∑–∞–ø—É—â–µ–Ω –≤ –Ω–µ—Å–∫–æ–ª—å–∫–∏—Ö —ç–∫–∑–µ–º–ø–ª—è—Ä–∞—Ö (–∫–æ–Ω—Ç–µ–π–Ω–µ—Ä–∞—Ö) –∏ –º–∞—Å—à—Ç–∞–±–∏—Ä–æ–≤–∞—Ç—å—Å—è –≥–æ—Ä–∏–∑–æ–Ω—Ç–∞–ª—å–Ω–æ –ø—Ä–∏ —Ä–æ—Å—Ç–µ –Ω–∞–≥—Ä—É–∑–∫–∏. –ù–∞–ø—Ä–∏–º–µ—Ä, –µ—Å–ª–∏ —Ä–∞—Å—Ç–µ—Ç —á–∏—Å–ª–æ –∑–∞–ø—Ä–æ—Å–æ–≤ ‚Äì –º–æ–∂–Ω–æ –∑–∞–ø—É—Å—Ç–∏—Ç—å –±–æ–ª—å—à–µ –∫–æ–ø–∏–π LLM-–º–æ–¥—É–ª—è –∑–∞ –±–∞–ª–∞–Ω—Å–∏—Ä–æ–≤—â–∏–∫–æ–º, –∏–ª–∏ –≤—ã–Ω–µ—Å—Ç–∏ RAG-—Ö—Ä–∞–Ω–∏–ª–∏—â–µ –Ω–∞ –æ—Ç–¥–µ–ª—å–Ω—ã–µ —É–∑–ª—ã –¥–ª—è —É–≤–µ–ª–∏—á–µ–Ω–∏—è –æ–±—ä–µ–º–∞ –ø–∞–º—è—Ç–∏. –ö–æ–º–º—É–Ω–∏–∫–∞—Ü–∏—è –º–µ–∂–¥—É —Å–µ—Ä–≤–∏—Å–∞–º–∏ –æ—Å—É—â–µ—Å—Ç–≤–ª—è–µ—Ç—Å—è –ø–æ —á–µ—Ç–∫–æ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã–º API (REST –∏–ª–∏ gRPC), —á—Ç–æ –ø–æ–∑–≤–æ–ª—è–µ—Ç –æ–±–Ω–æ–≤–ª—è—Ç—å –∏—Ö –Ω–µ–∑–∞–≤–∏—Å–∏–º–æ.

## –ü–ª–∞–Ω –º–æ–Ω–∏—Ç–æ—Ä–∏–Ω–≥–∞ –∏ –∞–ª–µ—Ä—Ç–∏–Ω–≥–∞

–î–ª—è –ø–æ–¥–¥–µ—Ä–∂–∞–Ω–∏—è –Ω–∞–¥–µ–∂–Ω–æ–π —Ä–∞–±–æ—Ç—ã Braindler –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –≤–Ω–µ–¥—Ä–∏—Ç—å –∫–æ–º–ø–ª–µ–∫—Å–Ω–æ–µ –º–æ–Ω–∏—Ç–æ—Ä–∏–Ω–≥-—Ä–µ—à–µ–Ω–∏–µ, –æ—Ö–≤–∞—Ç—ã–≤–∞—é—â–µ–µ –≤—Å–µ –∫–ª—é—á–µ–≤—ã–µ —Å–µ—Ä–≤–∏—Å—ã. –ú—ã –±—É–¥–µ–º –æ—Ç—Å–ª–µ–∂–∏–≤–∞—Ç—å –º–µ—Ç—Ä–∏–∫–∏ –ø—Ä–æ–∏–∑–≤–æ–¥–∏—Ç–µ–ª—å–Ω–æ—Å—Ç–∏, –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è —Ä–µ—Å—É—Ä—Å–æ–≤, –æ—à–∏–±–æ–∫, –∞ —Ç–∞–∫–∂–µ —Å–ø–µ—Ü–∏—Ñ–∏—á–µ—Å–∫–∏–µ –ø–æ–∫–∞–∑–∞—Ç–µ–ª–∏ –∫–∞—á–µ—Å—Ç–≤–∞ —Ä–∞–±–æ—Ç—ã RAG –∏ LLM. –°–∏—Å—Ç–µ–º–∞ –º–æ–Ω–∏—Ç–æ—Ä–∏–Ω–≥–∞ (–Ω–∞–ø—Ä–∏–º–µ—Ä, —Å–≤—è–∑–∫–∞ Prometheus + Grafana –¥–ª—è –º–µ—Ç—Ä–∏–∫ –∏ Alertmanager –¥–ª—è –æ–ø–æ–≤–µ—â–µ–Ω–∏–π) –±—É–¥–µ—Ç —Å–æ–±–∏—Ä–∞—Ç—å –¥–∞–Ω–Ω—ã–µ –∏ —Å–∏–≥–Ω–∞–ª–∏–∑–∏—Ä–æ–≤–∞—Ç—å –æ –æ—Ç–∫–ª–æ–Ω–µ–Ω–∏—è—Ö. –ù–∏–∂–µ –ø–µ—Ä–µ—á–∏—Å–ª–µ–Ω—ã –æ—Å–Ω–æ–≤–Ω—ã–µ —ç–ª–µ–º–µ–Ω—Ç—ã –ø–ª–∞–Ω–∞ –º–æ–Ω–∏—Ç–æ—Ä–∏–Ω–≥–∞:

- **–ü—Ä–æ–∏–∑–≤–æ–¥–∏—Ç–µ–ª—å–Ω–æ—Å—Ç—å –∏ –≤—Ä–µ–º—è –æ—Ç–∫–ª–∏–∫–∞:** –û—Ç—Å–ª–µ–∂–∏–≤–∞–µ—Ç—Å—è *Latency* –∫–∞–∂–¥–æ–≥–æ —ç—Ç–∞–ø–∞:
  - –í—Ä–µ–º—è –æ—Ç–≤–µ—Ç–∞ –±–æ—Ç–∞ (–æ—Ç –ø–æ–ª—É—á–µ–Ω–∏—è —Å–æ–æ–±—â–µ–Ω–∏—è –¥–æ –æ—Ç–ø—Ä–∞–≤–∫–∏) ‚Äì –∫–ª—é—á–µ–≤–æ–π –ø–æ–∫–∞–∑–∞—Ç–µ–ª—å UX.
  - –í—Ä–µ–º—è –æ–±—Ä–∞–±–æ—Ç–∫–∏ –∑–∞–ø—Ä–æ—Å–∞ –Ω–∞ backend (end-to-end –ª–∞—Ç–µ–Ω—Ç–Ω–æ—Å—Ç—å).
  - –í—Ä–µ–º—è –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è –ø–æ–¥–∑–∞–ø—Ä–æ—Å–æ–≤: –ø–æ–∏—Å–∫ RAG (–¥–æ–ª–∂–Ω–æ –±—ã—Ç—å –º–∏–ª–ª–∏—Å–µ–∫—É–Ω–¥—ã –¥–ª—è –Ω–µ–±–æ–ª—å—à–æ–≥–æ –æ–±—ä–µ–º–∞, —É–≤–µ–ª–∏—á–∏–≤–∞–µ—Ç—Å—è —Å —Ä–æ—Å—Ç–æ–º –±–∞–∑—ã) –∏ –≥–µ–Ω–µ—Ä–∞—Ü–∏—è –æ—Ç–≤–µ—Ç–∞ LLM (–∑–∞–≤–∏—Å–∏—Ç –æ—Ç –¥–ª–∏–Ω—ã –æ—Ç–≤–µ—Ç–∞ –∏ —Ä–∞–∑–º–µ—Ä–∞ –º–æ–¥–µ–ª–∏).  
  *–ú–µ—Ç—Ä–∏–∫–∏:* `request_duration_seconds` (–≥–∏—Å—Ç–æ–≥—Ä–∞–º–º–∞/—Å—Ä–µ–¥–Ω–µ–µ), –æ—Ç–¥–µ–ª—å–Ω–æ –ø–æ —Ç–∏–ø–∞–º –∑–∞–ø—Ä–æ—Å–æ–≤; `rag_search_time_ms`; `llm_inference_time_ms`.  
  *–ê–ª–µ—Ä—Ç—ã:* –µ—Å–ª–∏ –≤—Ä–µ–º—è –æ—Ç–≤–µ—Ç–∞ –ø—Ä–µ–≤—ã—à–∞–µ—Ç –∑–∞–¥–∞–Ω–Ω—ã–π –ø–æ—Ä–æ–≥ (–Ω–∞–ø—Ä–∏–º–µ—Ä, >3 —Å–µ–∫ –¥–ª—è –æ–±—ã—á–Ω–æ–≥–æ –≤–æ–ø—Ä–æ—Å–∞) –≤ —Ç–µ—á–µ–Ω–∏–µ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω–æ–≥–æ –∫–æ–ª–∏—á–µ—Å—Ç–≤–∞ –∑–∞–ø—Ä–æ—Å–æ–≤ ‚Äì –æ—Ç–ø—Ä–∞–≤–ª—è—Ç—å –ø—Ä–µ–¥—É–ø—Ä–µ–∂–¥–µ–Ω–∏–µ. –≠—Ç–æ –º–æ–∂–µ—Ç —É–∫–∞–∑—ã–≤–∞—Ç—å –Ω–∞ –ø–µ—Ä–µ–≥—Ä—É–∑–∫—É –∏–ª–∏ –ø—Ä–æ–±–ª–µ–º—ã –≤ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–µ–º –º–æ–¥—É–ª–µ.

- **–ù–∞–≥—Ä—É–∑–∫–∞ –∏ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ —Ä–µ—Å—É—Ä—Å–æ–≤:** –í–∞–∂–Ω—ã –ø–æ–∫–∞–∑–∞—Ç–µ–ª–∏ —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ–≥–æ —Ä–∞—Å—Ö–æ–¥–æ–≤–∞–Ω–∏—è –Ω–∞—à–∏—Ö GPU –∏ —Å–µ—Ä–≤–µ—Ä–æ–≤:
  - **GPU**: –∑–∞–≥—Ä—É–∑–∫–∞ GPU (% –∑–∞–Ω—è—Ç–æ—Å—Ç–∏ —è–¥—Ä–∞), –ø–æ—Ç—Ä–µ–±–ª–µ–Ω–∏–µ –ø–∞–º—è—Ç–∏ –≤–∏–¥–µ–æ–∫–∞—Ä—Ç—ã. –ú–æ–¥–µ–ª—å LLM –¥–æ–ª–∂–Ω–∞ –ø–æ–º–µ—â–∞—Ç—å—Å—è –≤ –ø–∞–º—è—Ç—å; –µ—Å–ª–∏ –≤–∏–¥–∏–º –±–ª–∏–∑–∫–æ –∫ 100% –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–∏ –ø–∞–º—è—Ç–∏, –µ—Å—Ç—å —Ä–∏—Å–∫ –æ—à–∏–±–æ–∫ ‚Äì –Ω—É–∂–µ–Ω –∞–ª–µ—Ä—Ç. –ó–∞–≥—Ä—É–∑–∫–∞ GPU –ø–æ–∫–∞–∂–µ—Ç, —Ö–≤–∞—Ç–∞–µ—Ç –ª–∏ –º–æ—â–Ω–æ—Å—Ç–∏ –∏–ª–∏ –º–æ–¥–µ–ª—å —Ç–æ—Ä–º–æ–∑–∏—Ç (–µ—Å–ª–∏ –ø–æ—Å—Ç–æ—è–Ω–Ω–æ 100%, –Ω–∞–¥–æ –º–∞—Å—à—Ç–∞–±–∏—Ä–æ–≤–∞—Ç—å).
  - **CPU –∏ RAM**: –Ω–∞ —É–∑–ª–∞—Ö backend –∏ RAG ‚Äì CPU_load, memory_usage. –í–µ–∫—Ç–æ—Ä–Ω—ã–π –ø–æ–∏—Å–∫ –∏ —Å–∞–º–∞ LLM —Ç–∞–∫–∂–µ –º–æ–≥—É—Ç —á–∞—Å—Ç–∏—á–Ω–æ –≥—Ä—É–∑–∏—Ç—å CPU (–ø—Ä–µ–¥–æ–±—Ä–∞–±–æ—Ç–∫–∞, –ø–æ—Å—Ç–æ–±—Ä–∞–±–æ—Ç–∫–∞).
  - **–î–∏—Å–∫ –∏ I/O**: –º–æ–Ω–∏—Ç–æ—Ä–∏–Ω–≥ –¥–∏—Å–∫–æ–≤–æ–≥–æ –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–∞ (–æ—Å–æ–±–µ–Ω–Ω–æ –¥–ª—è —Ö—Ä–∞–Ω–µ–Ω–∏—è –¥–æ–∫—É–º–µ–Ω—Ç–æ–≤, –∏–Ω–¥–µ–∫—Å–æ–≤) –∏ —Å–∫–æ—Ä–æ—Å—Ç–∏ I/O, –µ—Å–ª–∏ –±–∞–∑–∞ –∑–Ω–∞–Ω–∏–π –±–æ–ª—å—à–∞—è.
  - **–°–µ—Ç–µ–≤—ã–µ –º–µ—Ç—Ä–∏–∫–∏**: —Ç—Ä–∞—Ñ–∏–∫ –º–µ–∂–¥—É –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–∞–º–∏, –æ—Å–æ–±–µ–Ω–Ω–æ –µ—Å–ª–∏ –æ–Ω–∏ –Ω–∞ —Ä–∞–∑–Ω—ã—Ö —É–∑–ª–∞—Ö. –í–∞–∂–Ω–∞ –ø—Ä–æ–ø—É—Å–∫–Ω–∞—è —Å–ø–æ—Å–æ–±–Ω–æ—Å—Ç—å –∏ –∑–∞–¥–µ—Ä–∂–∫–∏ —Å–µ—Ç–∏, —á—Ç–æ–±—ã RAG –∏ LLM –æ—Ç–≤–µ—Ç—ã –±—ã—Å—Ç—Ä–æ –¥–æ—Å—Ç–∏–≥–∞–ª–∏ backend.  
  *–ú–µ—Ç—Ä–∏–∫–∏:* `gpu_utilization_percent`, `gpu_memory_bytes_used`, `cpu_usage_percent`, `memory_rss_bytes`, `disk_free_bytes`, `network_latency_ms` (–º–µ–∂—Å–µ—Ä–≤–∏—Å–Ω–∞—è, –µ—Å–ª–∏ –≤–æ–∑–º–æ–∂–Ω–æ –∑–∞–º–µ—Ä—è—Ç—å).  
  *–ê–ª–µ—Ä—Ç—ã:* –ø—Ä–∏ –ø—Ä–µ–≤—ã—à–µ–Ω–∏–∏ –∫—Ä–∏—Ç–∏—á–µ—Å–∫–∏—Ö –ø–æ—Ä–æ–≥–æ–≤ ‚Äì –Ω–∞–ø—Ä–∏–º–µ—Ä, CPU > 85% –ø–æ—Å—Ç–æ—è–Ω–Ω–æ –≤ —Ç–µ—á–µ–Ω–∏–µ 5 –º–∏–Ω—É—Ç, —Å–≤–æ–±–æ–¥–Ω–∞—è RAM < 10%, –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–æ –Ω–∞ –¥–∏—Å–∫–µ < 15% ‚Äì —Å–æ–∑–¥–∞–≤–∞—Ç—å –∏–Ω—Ü–∏–¥–µ–Ω—Ç. –ê —Ç–∞–∫–∂–µ –æ—Ç–¥–µ–ª—å–Ω—ã–π –∞–ª–µ—Ä—Ç, –µ—Å–ª–∏ **GPU memory** –∑–∞–ø–æ–ª–Ω–µ–Ω–∞ >90% (–º–æ–∂–µ—Ç —Å–∫–æ—Ä–æ OOM).

- **–°—Ç–∞–±–∏–ª—å–Ω–æ—Å—Ç—å –∏ –æ—à–∏–±–∫–∏:** –û—Ç—Å–ª–µ–∂–∏–≤–∞–µ–º –∫–æ—Ä—Ä–µ–∫—Ç–Ω–æ—Å—Ç—å —Ä–∞–±–æ—Ç—ã —Å–µ—Ä–≤–∏—Å–æ–≤:
